# **Node.js 모듈**

**모듈**이란 특정한 기능을 하는 함수나 변수들의 집합이다.

모듈을 만들면 여러 프로그램에 해당 모듈을 재사용할 수 있다.

> 모듈 만들기

<pre><code>module.export = { 객체or함수or변수 };</code></pre>

> 모듈 불러오기

<pre><code>require('모듈의 경로');</code></pre>


**여러 파일에 걸쳐 재사용되는 함수나 변수들을 모듈로 만들어두면 편리하다.**
