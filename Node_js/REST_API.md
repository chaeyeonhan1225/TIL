# **REST API** 

REpresentational State Transfer의 약어

서버의 자원을 정의하고, 자원에 대한 주소를 지정하는 방법을 가리킨다.

> **HTTP 요청 메서드**

- **GET** : 서버 자원을 가져오고자 할 때 사용
- **POST** : 서버에 자원을 새로 등록
- **PUT** : 서버의 자원을 대체
- **DELETE** : 서버의 자원을 삭제

REST API를 따르는 서버를 RESTful하다고 표현한다.
